# Finanza Zuid

A modern financial services platform built with Next.js.

## Features
- 🎨 Responsive design with Tailwind CSS
- 🔐 Authentication with Auth0
- 👥 Member management
- 📅 Event registration
- 📚 Resource library
- 📝 Contact forms

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Auth0
- Supabase
- Framer Motion

## Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Environment Variables
Create a `.env.local` file with:
```
AUTH0_SECRET=
AUTH0_BASE_URL=
AUTH0_ISSUER_BASE_URL=
AUTH0_CLIENT_ID=
AUTH0_CLIENT_SECRET=
```

## Production
```bash
# Build for production
npm run build

# Start production server
npm start
```
